# Audio-transcribator

Сервис для обработки аудио- и видеофайлов: загрузка файла на сервер, извлечение аудио из видео, транскрибация, генерация summary и скачивание результатов.

## Что уже реализовано

### 1. Backend API

Реализован FastAPI-сервис с основными endpoint'ами:

- `POST /login` — авторизация пользователя и получение токена;
- `POST /process` — загрузка аудио/видео файла и запуск обработки;
- `GET /result/{job_id}` — получение результата обработки по ID задачи;
- `GET /download/{job_id}/{filename}` — скачивание отдельных файлов результата.

### 2. Авторизация

Реализована базовая token-based авторизация:

- пользователь вводит логин и пароль;
- сервер возвращает access token;
- дальнейшие запросы выполняются с заголовком `Authorization: Bearer <token>`.

Текущая реализация подходит для MVP. Далее можно расширить до JWT, нескольких пользователей и разграничения доступа к задачам.

### 3. Обработка файлов

Поддерживается загрузка:

- аудио: `.mp3`, `.wav`, `.m4a` и др.;
- видео: `.mp4`, `.mov`, `.mkv`, `.webm` и др.

Для видео автоматически извлекается аудиодорожка через `ffmpeg`.

### 4. Транскрибация

Для распознавания речи используется `faster-whisper`.

Текущая модель: `Whisper base`.

audio/video file
→ audio extraction via ffmpeg
→ transcription via faster-whisper
→ transcript.txt
